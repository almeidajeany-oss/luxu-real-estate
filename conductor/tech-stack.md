# Tech Stack: luxu-real-estate

## Core Framework & Language
- **Next.js 16.2.0 (App Router)**: Our primary React framework, chosen for its advanced routing features, server components, and performance optimizations.
- **React 19.2.4**: The library for building our user interfaces, integrated with Next.js.
- **TypeScript**: Our programming language, providing strong typing and enhanced developer experience for a more robust codebase.

## Styling & UI
- **Tailwind CSS 4**: A utility-first CSS framework for rapid and consistent styling.
- **Modern UI Patterns**: Focus on responsive, accessible, and performant UI components that align with luxury real estate standards.

## Architecture
- **Next.js App Router**: Utilizing Server Components and modern data fetching patterns to ensure a performant and scalable application structure.
